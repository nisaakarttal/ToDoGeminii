# 🚀 ToDoGeminii

A high-performance, asynchronous To-Do API built with **FastAPI** and **Python 3.14**. Powered by **SQLAlchemy** and **Pydantic**, featuring secure **JWT-based authentication** and ready-to-use **Swagger/Redoc** documentation.

---

## ✨ Features
* **Fast & Modern:** Built with the latest Python 3.14 features.
* **Secure:** JWT-based user login and registration system.
* **Database:** Robust data management using SQLAlchemy ORM.
* **Auto-Docs:** Interactive API documentation via Swagger UI.

## 🛠️ Tech Stack
* **Language:** Python 3.14
* **Framework:** FastAPI
* **ORM:** SQLAlchemy
* **Validation:** Pydantic
* **Auth:** Python-JOSE (JWT)
